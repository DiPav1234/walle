# 🤍 Define Dancing — WALL·E + EVE

Сторінка-сюрприз: ВАЛЛ·І та EVE кружляють у вальсі серед зірок.

**Live:** https://dipav1234.github.io/walle/ · noindex

Один самодостатній `index.html` (vanilla canvas, без збірки).
Налаштування — блок `CONFIG` на початку `<script>`:
- `MESSAGE` — особистий підпис;
- `INITIALS` — букви в сердечку, яке ВАЛЛ·І дарує EVE;
- `DATE_TEXT` — особлива дата сузір'ям у небі.

Клік — вальс закружляє швидше + сплеск сердець. ♪ — ніжна музика.
